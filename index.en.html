Lab 1 — XML basics

XML is used to organize information. A well-formed file looks like a tree: one root, closed tags and correctly nested branches. A valid file also follows a grammar: here, a DTD.

One DTD for the forest

The DTD below accepts a forest, trees, trunks, branches and leaves. It validates the four corrected files.

<!ELEMENT foret (#PCDATA|arbre)*>
<!ELEMENT arbre (tronc)>
<!ELEMENT tronc (branche*)>
<!ELEMENT branche (branche|feuille)*>
<!ELEMENT feuille EMPTY>

Validation and engineering quality

The portfolio includes a checklist: W3C validators, well-formed XML, DTD/XSD, XSLT, accessibility, HTML language, contrasts, metadata and source files.

Checklist

  • HTML5 / XHTML polyglotte : balises fermées et attributs quotés.
  • Accessibilité : contraste, texte alternatif, lien d'évitement, langues déclarées.
  • Référencement : hreflang, schema.org, titres et descriptions.
  • Sources : XML, DTD, XSD, XSLT, SVG, RDF/Turtle, SPARQL.
