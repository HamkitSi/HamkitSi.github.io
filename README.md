# Portfolio Web sémantique — Hasina RAKOTOARISON

Ce dépôt contient mon portfolio de Web sémantique, réalisé pour présenter mon profil, mon CV et mes travaux de TP.

Le site est disponible en trois langues : français, anglais et japonais. Il est organisé autour d’un thème visuel de forêt, avec des couleurs beige, vert et marron.

## Contenu principal

- `index.html`, `index.en.html`, `index.ja.html` : pages d’accueil multilingues.
- `cv-rdfa.html`, `cv-rdfa.en.html`, `cv-rdfa.ja.html` : CV multilingue annoté en RDFa.
- `data/` : source XML multilingue et schéma XSD.
- `tp1/` : fichiers du TP XML, DTD, XPath, CSS et SVG.
- `xml-xslt/` : travaux sur XML Schema et XSLT.
- `rdf-corese/` : modèle RDF et requêtes SPARQL pour Corese.
- `rdfa/` : triplets Turtle attendus, requêtes SPARQL et méthode d’extraction RDFa.
- `docs/` : comptes rendus des TP réalisés.

## Technologies utilisées

HTML5, CSS, XML, DTD, XSD, XPath, XSLT, SVG, RDF, RDFS, SPARQL, RDFa et GitHub Pages.
